# Ejecutar
Para ejecutar, clonamos el repositorio y para que puede correr sobre el servidor Jetty:

```sh
$ git clone https://github.com/Rebecasarai/PivotalSimpleProject.git
$ cd PivotalSimpleProject
$ mvn package
$ mvn jetty:run
```
